# 🔢 Number Generator

A simple, interactive web application that generates random numbers within a specified range. This project marks my first step into JavaScript development!

## 🚀 Features
- **Random Generation:** Instantly generates a random number at the click of a button.
- **Clean UI:** Styled with CSS for a modern, user-friendly experience.
- **Responsive Design:** Works on both desktop and mobile screens.

## 🛠️ Technologies Used
- **HTML5:** For the structure of the web page.
- **CSS3:** For styling and layout.
- **JavaScript (ES6):** For the logic and interactivity.
